Directory structure
===================

When HPhi-xxx.tar.gz is unzipped, the following directory structure is composed.
::

 |--CMakeLists.txt
 |--COPYING
 |--config/
 |    |--fujitsu.cmake
 |    |--gcc.cmake
 |    |--intel.cmake
 |    ---sekirei.cmake
 |--doc/
 |    |--en/
 |    |--jp/
 |    |--fourier/
 |    |    |--en/
 |    |    |--figs/
 |    |    ---ja/
 |    |--userguide_en.pdf
 |    ---userguide_jp.pdf
 |--HPhiconfig.sh
 |--samples/
 |--src/
 |    |--*.c
 |    |--CMakeLists.txt
 |    |--include/*.h
 |    |--makefile_src
 |    ---StdFace/
 |--test/
 ---test_tool/
